# rainbow/

VBS files for keyboard shortcut profiles, and the hotkey daemon.

## Purpose

Contains files generated by `setup.ps1` for the keyboard shortcuts feature:
- One VBS per shortcut slot (calls OpenRGB with a specific profile)
- `hotkeys.ps1` — the hotkey daemon (registered via Win32 `RegisterHotKey`)
- `hotkeys_runner.vbs` — silent launcher for `hotkeys.ps1`

## Contents

Files are auto-generated from `config.json` (`shortcuts` section) when `setup.ps1` runs:

| File | Description |
|------|-------------|
| `{key}.vbs` | One per assigned shortcut (e.g., `F1.vbs`, `1.vbs`) |
| `hotkeys.ps1` | Hidden PowerShell hotkey daemon |
| `hotkeys_runner.vbs` | Launches `hotkeys.ps1` silently |

For current assignments, see [config.json](../config.json).

## Usage

`hotkeys.ps1` runs automatically at login via the `OpenRGB hotkeys` Task Scheduler task.
It registers global hotkeys (modifier + key) and dispatches them to OpenRGB.

The individual `.vbs` files exist for standalone use but are not called by the daemon.

## Notes

- Files are NOT edited manually - edit `config.json` then run `setup.ps1`
- If OpenRGB server is running at time of hotkey press: instant profile change
- If server is not running: ~2-3 sec delay (OpenRGB starts up)
- Daemon starts immediately after `setup.ps1` completes (no reboot needed)
